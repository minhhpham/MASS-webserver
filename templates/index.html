WELCOME!


This is a web-based tool that can help civil engineers and policymakers make optimal decisions in developing water reclamation and reuse chain. The tool will ask for information about population clusters under consideration and potential water-treatment locations, then use the provided information to mathematically minimize the costs and environmental impacts and to maximize the value of water recovery. Finally, the recommended designs are presented through intelligible visualizations.

To view a quick demonstration, please take a tour at Demo.

For guidance to use this tool, please read the Documentation.

If you have any question, please check the Contact page.

To start using the tool, please log in.

This tool is free and open-source. It was supported by the NSF #1454559.

Browser Compatibility: We have tested that this site functions well on Google Chrome, Mozilla FireFox, Microsoft Edge, and Safari. Microsoft Internet Explorer may or may not work.
